# CRAFT Website — Content Drop Folder

This is where you drop sample content for the new site. Anything you put in here, I'll use when building in Webflow. The folders below match the site's content needs — drop things into the most-relevant folder and don't worry about file naming, I'll rename and organize.

---

## 01-photos/
The big one. The site needs real photography to look like AirNoss instead of a template. Drop anything you have — phone photos are fine for placeholders, but flag if it's professional vs. casual.

- **fleet/** — DA40NG and DA42-VI photos. Exterior, interior, cockpit, panel shots. Wide angles for hero use.
- **fbo-location/** — Atlantic FBO ramp shots, KCHS terminal, the CRAFT hangar/office, signage, ramp activity.
- **instructors/** — Headshots and on-the-ramp shots of CFIs. Names + bios in `06-testimonials/` if available.
- **sim/** — The Redbird FMX in the room, students in the sim, panel close-ups. This is the wedge — needs to look serious.
- **students-cockpit/** — Lifestyle shots: students in the left seat, instructor in the right, post-solo grins.
- **events/** — Open houses, ground school sessions, anything community-feeling.

## 02-copy/
Anything written. Drop as .docx, .md, .txt, .pdf — whatever you have.
- Existing program descriptions
- Pricing sheets (real numbers — replace the placeholders in the prototype)
- About / company history
- FAQ that gets asked a lot
- The CRAFT one-liner / mission if there is one

## 03-brand-assets/
- Logos (you already have `Craft-logo-FINAL.png` in the parent folder — that's fine)
- Any existing brand guide (color codes, font preferences)
- Letterhead, business card files, anything that hints at the brand system

## 04-existing-content/
- Screenshots or scrapes of the current `flycraftchs.com` site if there's content I need to preserve (testimonials, instructor bios, location info, etc.)
- Old marketing materials, brochures, postcards

## 05-program-details/
The accelerated program info specifically. This drives the Programs CMS in Webflow.
- For each of: Discovery Flight, Accelerated PPL, Zero-to-Commercial, Accelerated IFR, CFI Add-on, Multi-Engine, Part 61 PPL
  - Real all-in price
  - Real timeline (weeks/months)
  - Hours breakdown (flight hrs, sim hrs, instructor hrs)
  - What's included / not included
  - Prerequisites
  - Next start date

## 06-testimonials/
- Student names + ratings achieved + program completed + a quote (even rough)
- Permission flag (OK to use real name? OK to use photo?)
- Where they ended up after CRAFT (Part 121 regional? Part 135? Corporate? CFI?)

## 07-reference/
- Inspiration screenshots beyond AirNoss
- Things you specifically don't want (anti-references)
- Notes / random ideas / scribbles
