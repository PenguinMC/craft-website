# CRAFT Email Templates — v2

These are full responsive HTML email bodies (table-based, dark-theme, beacon-red accents). Each is designed to be pasted directly into HubSpot's email rich-text editor.

## How to use
1. HubSpot → Marketing → Email
2. Open the draft email matching the filename
3. Click into the empty content body
4. Toggle "Source" / "View source" mode
5. Paste the entire HTML
6. Save → Review and publish → Save for automation
